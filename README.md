# eHR Project Webapp 


## Help the project
__Install__

    一、在Eclipse中导该项目


    二、创建MySQL数据库ehr，并导入data/ehr.sql数据


    三、修改 src\main\resources\applicationContext-root.xml 中的数据库连接


    四、发布并运行（默认为localhost:8080/ehr-webapp）
    
    五、按发布情况访问
     http://localhost:8080/ehr-webapp
     使用 admin/admin123 登陆
